# 🌐 Manisha Developer — Portfolio Website

A personal portfolio website built with pure HTML, CSS & JavaScript — designed for Fiverr freelancing. Features a 3D animated hero card, scroll-driven animations, marquee tickers, and a fully responsive light theme.

---

## ✨ Features

- 🃏 3D hero card with mouse-tracking rotation and scroll parallax
- 🎬 Scroll-driven card animations (slide from left, right, rotate-in)
- 🎠 3 marquee ticker strips between sections
- 💡 Typewriter effect cycling through roles
- 📊 Animated skill bars and stat counters
- 🌟 Shine sweep + 3D tilt on all cards
- 🎯 Scroll progress bar at top of page
- 📱 Fully responsive on mobile, tablet & desktop
- ✨ Cursor glow trail effect

---

## 🛠️ Built With

- HTML5
- CSS3 (animations, keyframes, 3D transforms)
- Vanilla JavaScript
- Google Fonts — Plus Jakarta Sans

---

## 📂 Sections

- **Hero** — Intro with 3D animated profile card
- **Services** — 6 freelance services offered
- **Projects** — 8 real GitHub projects
- **Experience** — 4 internships timeline
- **Skills** — Animated skill bars
- **Certifications** — 8 certificates & achievements
- **Contact** — Email, phone, GitHub, location
